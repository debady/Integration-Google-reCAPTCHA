
 Nom du projet :  
Integration-Google-reCAPTCHA

     Description :  
Integration-Google-reCAPTCHA est un projet simple qui montre comment intégrer Google reCAPTCHA dans un formulaire HTML. Ce dépôt contient le code source nécessaire pour protéger les formulaires web contre les bots et les abus automatisés.

     Fonctionnalités principales :  
- Validation reCAPTCHA v2 : Intègre la vérification visuelle de Google (case \"Je ne suis pas un robot\").  
- Traitement côté serveur : Validation des réponses reCAPTCHA avec une clé secrète.  
- Exemple complet : Inclut le formulaire HTML, le style CSS et le script PHP de validation.

     Technologies utilisées :  
- Langages : HTML, CSS, PHP.  
- API utilisée : Google reCAPTCHA v2.  

     Installation :  
1. Clonez le dépôt :  
   
   bash
   git clone https://github.com/votre_utilisateur/Integration-Google-reCAPTCHA.git
   

2. Accédez au dossier du projet :  
   
   bash
   cd Integration-Google-reCAPTCHA
   

3. Configurez vos clés reCAPTCHA :  
   - Ouvrez le fichier `index.html` et remplacez `VOTRE_SITE_KEY` par votre clé publique.  
   - Ouvrez le fichier `validate.php` et remplacez `VOTRE_SECRET_KEY` par votre clé secrète.  

4. Exécutez le projet en local avec un serveur comme XAMPP ou WAMP.
